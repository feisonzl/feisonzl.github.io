[原]shell 初识

1.shell简介

1.1 shell

Shell 是一个用<span class="hljs-keyword">C</span>语言编写的程序，它是用户使用Linux的桥梁。Shell既是一种命令语言，又是一种程序设计语言。
Shell 是指一种应用程序，这个应用程序提供了一个界面，用户通过这个界面访问操作系统内核的服务。`</pre>

#### 1.2 shell脚本

<pre class="prettyprint">`Shell 脚本（<span class="hljs-built_in">shell</span> script），是一种为<span class="hljs-built_in">shell</span>编写的脚本程序。`</pre>

#### 1.3shell类型

<pre class="prettyprint">`bash
csh
dash
ksh
<span class="hljs-attribute">...</span><span class="hljs-built_in">.</span>`</pre>

### 2.shell入门

#### 2.1编写第一个shell脚本

<pre class="prettyprint">`test.sh:
<span class="hljs-shebang">#!/bin/bash</span>
<span class="hljs-built_in">echo</span> <span class="hljs-string">"Hello World !"</span>

注：
    <span class="hljs-comment">#!为一个约定的标记，指定shell的相应类型，用什么解释器执行</span>`</pre>

#### 2.2执行脚本

<pre class="prettyprint">`<span class="hljs-number">1.</span>
    <span class="hljs-preprocessor">#chmod +x test.sh</span>
    <span class="hljs-preprocessor">#./test.sh</span>
<span class="hljs-number">2.</span>
    /bin/bash test<span class="hljs-preprocessor">.sh</span>
`</pre>

### 3.shell 变量

#### 3.1变量定义

<pre class="prettyprint">`<span class="hljs-built_in">var</span><span class="hljs-subst">=</span>val
<span class="hljs-built_in">var</span><span class="hljs-subst">=</span><span class="hljs-string">"val"</span>
<span class="hljs-built_in">var</span><span class="hljs-subst">=</span><span class="hljs-string">'val'</span>
循环赋值：
for file <span class="hljs-keyword">in</span> <span class="hljs-string">'ls /etc'</span>
规则：
<span class="hljs-number">1.</span>变量定义时，不使用$符号；
<span class="hljs-number">2.</span>变量名与等号之间不能有空格；
<span class="hljs-number">3.</span>首个字符必须为字母（a<span class="hljs-attribute">-z</span>，A<span class="hljs-attribute">-Z</span>）；
<span class="hljs-number">4.</span>中间不能有空格，可以使用下划线（_）；
<span class="hljs-number">5.</span>不能使用标点符号；
<span class="hljs-number">6.</span>不能使用bash里的关键字（可用help命令查看保留关键字）。`</pre>

#### 3.2使用变量

<pre class="prettyprint">`<span class="hljs-keyword">var</span>=val
<span class="hljs-keyword">echo</span> <span class="hljs-variable">$var</span>
<span class="hljs-keyword">echo</span> $(<span class="hljs-keyword">var</span>)`</pre>

#### 3.3只读变量

<pre class="prettyprint">`通过<span class="hljs-keyword">readonly</span>关键字声明：
<span class="hljs-keyword">var</span>=val
<span class="hljs-keyword">readonly</span> <span class="hljs-keyword">var</span>`</pre>

#### 3.4删除变量

<pre class="prettyprint">`通过<span class="hljs-keyword">unset</span>关键字声明：
<span class="hljs-keyword">var</span>=val
<span class="hljs-keyword">unset</span> <span class="hljs-keyword">var</span>`</pre>

#### 3.5变量类型

<pre class="prettyprint">`环境变量： 所有的程序，包括<span class="hljs-built_in">shell</span>启动的程序，都能访问环境变量，有些程序需要环境变量来保证其正常运行。必要的时候<span class="hljs-built_in">shell</span>脚本也可以定义环境变量。

局部变量：局部变量在脚本或命令中定义，仅在当前<span class="hljs-built_in">shell</span>实例中有效，其他<span class="hljs-built_in">shell</span>启动的程序不能访问局部变量。

<span class="hljs-built_in">shell</span>变量：<span class="hljs-built_in">shell</span>变量是由<span class="hljs-built_in">shell</span>程序设置的特殊变量。<span class="hljs-built_in">shell</span>变量中有一部分是环境变量，有一部分是局部变量，这些变量保证了<span class="hljs-built_in">shell</span>的正常运行`</pre>

#### 3.6shell字符串

<pre class="prettyprint">`<span class="hljs-number">1</span>.字符串声明：
单引号：
    str=<span class="hljs-string">'string'</span>
    单引号里的任何字符都会原样输出，单引号字符串中的变量是无效的；
    单引号字串中不能出现单引号（对单引号使用转义符后也不行）。
双引号：
    str=<span class="hljs-string">"string"</span>
    双引号里可以有变量；
    双引号里可以出现转义字符
无引：
    str=string
<span class="hljs-number">2</span>.拼接字符串：
    your_name=<span class="hljs-string">"qinjx"</span>
    greeting=<span class="hljs-string">"hello, "</span><span class="hljs-variable">$your_name</span><span class="hljs-string">" !"</span>
    greeting_1=<span class="hljs-string">"hello, <span class="hljs-variable">${your_name}</span> !"</span>
    <span class="hljs-built_in">echo</span> <span class="hljs-variable">$greeting</span> <span class="hljs-variable">$greeting_1</span>
<span class="hljs-number">3</span>.获取字符串长度：
    string=<span class="hljs-string">"abcd"</span>
    <span class="hljs-built_in">echo</span> <span class="hljs-variable">${#string}</span> <span class="hljs-comment">#输出 4</span>
<span class="hljs-number">4</span>.提取子字符串：
    string=<span class="hljs-string">"runoob is a great site"</span>
    <span class="hljs-built_in">echo</span> <span class="hljs-variable">${string:1:4}</span> <span class="hljs-comment"># 输出 unoo</span>
<span class="hljs-number">5</span>.查找子字符串：
    string=<span class="hljs-string">"runoob is a great company"</span>
    <span class="hljs-built_in">echo</span> `expr index <span class="hljs-string">"<span class="hljs-variable">$string</span>"</span> is`  <span class="hljs-comment"># 输出 8</span>
    注意： 以上脚本中 <span class="hljs-string">"`"</span> 是反引号，而不是单引号 <span class="hljs-string">"'"</span>。`</pre>

#### 3.7shell数组

<pre class="prettyprint">`bash支持一维数组（不支持多维数组），并且没有限定数组的大小。
<span class="hljs-number">1.</span>定义数组
    array=(val0 val1 val2 <span class="hljs-keyword">...</span>)

    array_name=(
    value0
    value1
    value2
    value3
    )

    array_name[<span class="hljs-number">0</span>]=value0
    array_name[<span class="hljs-number">1</span>]=value1
    array_name[n]=valuen
<span class="hljs-number">2.</span>读取数组
    ${array[index]}
    通过<span class="hljs-string">"@"</span>获得数组中的所有元素：
    echo ${array[@]}
<span class="hljs-number">3.</span>获取数组长度
    获取数组元素个数：
    length=${<span class="hljs-comment">#array[@]}</span>
    length=${<span class="hljs-comment">#array[*]}</span>
    获得单个数组元素的长度：
    length=${<span class="hljs-comment">#array[n]}</span>`</pre>

#### 3.8shell注释

<pre class="prettyprint">`<span class="hljs-number">1.</span>以<span class="hljs-string">"#"</span>开头行表示<span class="hljs-built_in">shell</span>注释行
<span class="hljs-number">2.</span><span class="hljs-built_in">shell</span>无多行注释操作，只能每行行头添加<span class="hljs-string">"#"</span>`</pre>

### 4.Shell 传递参数

#### 4.1传递参数使用

<pre class="prettyprint">`我们可以在执行 Shell 脚本时，向脚本传递参数，脚本内获取参数的格式为：<span class="hljs-variable">$n</span>。n 代表一个数字，<span class="hljs-number">1</span> 为执行脚本的第一个参数，<span class="hljs-number">2</span> 为执行脚本的第二个参数，以此类推……
例：
test.sh:
<span class="hljs-shebang">#!/bin/bash</span>
<span class="hljs-built_in">echo</span> <span class="hljs-string">"Shell 传递参数实例！"</span>;
<span class="hljs-built_in">echo</span> <span class="hljs-string">"执行的文件名：<span class="hljs-variable">$0</span>"</span>;
<span class="hljs-built_in">echo</span> <span class="hljs-string">"第一个参数为：<span class="hljs-variable">$1</span>"</span>;
<span class="hljs-built_in">echo</span> <span class="hljs-string">"第二个参数为：<span class="hljs-variable">$2</span>"</span>;
<span class="hljs-built_in">echo</span> <span class="hljs-string">"第三个参数为：<span class="hljs-variable">$3</span>"</span>;

效果：
$ ./test.sh <span class="hljs-number">1</span> <span class="hljs-number">2</span> <span class="hljs-number">3</span>
Shell 传递参数实例！
执行的文件名：test.sh
第一个参数为：<span class="hljs-number">1</span>
第二个参数为：<span class="hljs-number">2</span>
第三个参数为：<span class="hljs-number">3</span>`</pre>

#### 4.2特殊参数

<pre class="prettyprint">`<span class="hljs-variable">$#</span> 传递到脚本的参数个数
$* 以一个单字符串显示所有向脚本传递的参数。
    如<span class="hljs-string">"$*"</span>用「<span class="hljs-string">"」括起来的情况、以"</span><span class="hljs-variable">$1</span> <span class="hljs-variable">$2</span> … <span class="hljs-variable">$n</span><span class="hljs-string">"的形式输出所有参数。
$$    脚本运行的当前进程ID号
$! 后台运行的最后一个进程的ID号
<span class="hljs-variable">$@</span> 与$*相同，但是使用时加引号，并在引号中返回每个参数。
    如"</span><span class="hljs-variable">$@</span><span class="hljs-string">"用「"</span>」括起来的情况、以<span class="hljs-string">"<span class="hljs-variable">$1</span>"</span> <span class="hljs-string">"<span class="hljs-variable">$2</span>"</span> … <span class="hljs-string">"<span class="hljs-variable">$n</span>"</span> 的形式输出所有参数。
$- 显示Shell使用的当前选项，与<span class="hljs-keyword">set</span>命令功能相同。
$? 显示最后命令的退出状态。<span class="hljs-number">0</span>表示没有错误，其他任何值表明有错误。

注：
$* 与 <span class="hljs-variable">$@</span> 区别：
相同点：都是引用所有参数。
不同点：只有在双引号中体现出来。假设在脚本运行时写了三个参数 <span class="hljs-number">1</span>、<span class="hljs-number">2</span>、<span class="hljs-number">3</span>，，则 <span class="hljs-string">" * "</span> 等价于 <span class="hljs-string">"1 2 3"</span>（传递了一个参数），而 <span class="hljs-string">"@"</span> 等价于 <span class="hljs-string">"1"</span> <span class="hljs-string">"2"</span> <span class="hljs-string">"3"</span>（传递了三个参数）。
例：
test.sh:
<span class="hljs-shebang">#!/bin/bash
</span>
<span class="hljs-built_in">echo</span> <span class="hljs-string">"-- \$* 演示 ---"</span>
<span class="hljs-keyword">for</span> i <span class="hljs-keyword">in</span> <span class="hljs-string">"$*"</span>; <span class="hljs-keyword">do</span>
    <span class="hljs-built_in">echo</span> <span class="hljs-variable">$i</span>
<span class="hljs-keyword">done</span>

<span class="hljs-built_in">echo</span> <span class="hljs-string">"-- \$@ 演示 ---"</span>
<span class="hljs-keyword">for</span> i <span class="hljs-keyword">in</span> <span class="hljs-string">"<span class="hljs-variable">$@</span>"</span>; <span class="hljs-keyword">do</span>
    <span class="hljs-built_in">echo</span> <span class="hljs-variable">$i</span>
<span class="hljs-keyword">done</span>

效果：
$ ./test.sh <span class="hljs-number">1</span> <span class="hljs-number">2</span> <span class="hljs-number">3</span>
-- $* 演示 ---
<span class="hljs-number">1</span> <span class="hljs-number">2</span> <span class="hljs-number">3</span>
-- <span class="hljs-variable">$@</span> 演示 ---
<span class="hljs-number">1</span>
<span class="hljs-number">2</span>
<span class="hljs-number">3</span>`</pre>

### 5.shell基本运算符

<pre class="prettyprint">`原生bash不支持简单的数学运算，但是可以通过其他命令来实现，例如 awk 和 expr，expr 最常用。
expr 是一款表达式计算工具，使用它能完成表达式的求值操作。
注意
1.使用的是反引号 <span class="hljs-smartquote">` 而不是单引号 '</span>；
2.表达式和运算符之间要有空格，例如 2<span class="hljs-code">+2 是不对的，必须写成 2 +</span> 2，这与我们熟悉的大多数编程语言不一样。
例：
test.sh:
#!/bin/bash

val=<span class="hljs-code">`expr 2 + 2`</span>
echo "两数之和为 : $val"`</pre>

#### 10.2 Here Document

    document
