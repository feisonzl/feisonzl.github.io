[转]搬瓦工网速优化net-speeder

下载并解压源文件：

<span class="hljs-attribute">:~# wget https://github.com/snooda/net-speeder/archive/master.zip</span>
:~# unzip master.zip`</pre>

### 安装编译环境：

<pre class="prettyprint">`<span class="hljs-symbol">:~</span><span class="hljs-comment"># apt-get install libnet1-dev</span>
<span class="hljs-symbol">:~</span><span class="hljs-comment"># apt-get install libpcap0.8-dev </span>`</pre>

### 编译：

<pre class="prettyprint">`<span class="hljs-constant">Linux</span> <span class="hljs-constant">Cooked</span> interface使用编译（venetX，<span class="hljs-constant">OpenVZ</span>）：
<span class="hljs-symbol">:~</span><span class="hljs-comment"># sh build.sh -DCOOKED</span>

普通网卡使用编译（<span class="hljs-constant">Xen</span>，<span class="hljs-constant">KVM</span>，物理机）：
<span class="hljs-symbol">:~</span><span class="hljs-comment"># sh build.sh</span>`</pre>

### 使用：

<pre class="prettyprint">`需要root权限启动
<span class="hljs-preprocessor">#参数：./net_speeder 网卡名 加速规则（bpf规则）</span>
<span class="hljs-preprocessor">#ovz用法(加速所有ip协议数据)： </span>
:~# ./net_speeder venet0 <span class="hljs-string">"ip"</span>`</pre>

### 添加到开机启动项：

<pre class="prettyprint">`<span class="hljs-symbol">:~</span><span class="hljs-comment"># cp ./net_speeder /usr/bin</span>
<span class="hljs-symbol">:~</span><span class="hljs-comment"># echo -e 'nohup /usr/bin/net_speeder venet0 "ip" &gt;/dev/null 2&gt;&amp;1 &amp;\nexit 0' &gt;&gt; /etc/rc.local</span>
